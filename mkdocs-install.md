# MkDocs Quick Start
## Install program 

    pip install mkdocs

## Crete new project

    mkdocs new my-project

## Install theme
For example material theme:

    pip install mkdocs-material

Append the following line to mkdocs.yml:

    theme:
        name: 'material'

## Serve project

    mkdocs serve

go to `127.0.0.1:8000/`

## Add documents
Add documents into `/docs`
